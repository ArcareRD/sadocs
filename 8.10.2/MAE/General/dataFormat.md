# 資料型態

* ## Web支援版本
  
      8.9.2

* ## APP 支援版本

      008.009.000.001 以上(含)

* __資料型態__
  * __位元__
    * 預設值
          null
    * 說明
          只用0或1來存放資料
  * __文字__
    * 全唯碼
    * 預設值
          空字串("")
    * 說明
          以文字來存放資料
  * __數字__
    * 整數(bingint)
    * 整數(int)
    * 整數(smallint)
    * 整數(tinyint)
    * 數字&小數位=有
    * 預設值
          0
    * 說明
          以數字(含小數)來存放資料
  * __日期__
    * 日期
    * 日期時間
    * 預設值
          null
    * 說明
          以"2000-01-01 00:00:00"格式來存放資料
  * __邏輯__
    * 預設值
          null
    * 說明
          以"true"或"false"以及0或1格式來存放資料
