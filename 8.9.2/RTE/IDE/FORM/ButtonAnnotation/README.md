### <div id="sysbtn">系統按鈕</div>
* **保留系統按鈕功能**
	>`僅針對新增存回、修改存回、刪除 時有效`</br>
	>`有勾選時，執行新增存回、修改存回、刪除 會直接影響各檔區主要資料表的資料`</br>
	>`未勾選時，則須自行設計資料交易去異動所要影響的資料表`
	* **是**	
		* 新增存回
		* 修改存回
		* 刪除
	* 否：<font color=#ff0000>`系統按鍵將無作用，須自行設計功能`</font>

* **使用系統提示訊息盒**
	>`新增存回、修改存回、刪除是否顯示系統提示訊息`
	* 是
	* 否

* **自訂功能-延續執行**
	>`當新增存回、修改存回、刪除有其他內鍵，除了資料交易會在存回時同時執行，其餘都是存回後才會執行`</br>
	>`匯入程式會判斷當內鍵類型=3.資料交易時，內鍵執行時機為1.同時，其餘皆給2.延續`
	* 新增存回-延續執行
	* 修改存回-延續執行
	* 刪除-延續執行

### <div id="chkrecord">勾選記錄</div>

* **檔區**

* **使用前存入資料表**
	>`將勾選的資料，寫入<資料表.表單勾選記錄檔>暫存表格(SYS_TEMP_EF000000000248)。`
	* 是
	* <font color=#a6a6a6>否</font>

* **使用後清空資料表**
	>`將<資料表.表單勾選記錄檔>暫存表格(SYS_TEMP_EF000000000248)的資料刪除。`
	* 是
	* <font color=#a6a6a6>否</font>
		> `須自行設計"資料交易"刪除資料。但因IDE目前無法設計刪除暫存表格，所以資料會永遠保存在資料庫裡。`

* **同時清空勾選記錄**
	>`限制：當<使用後清空資料表=是>時，才可設定`</br>
	>`(#7858) 清除勾選資料。但畫面不會自動重整，須自行設計<重顯網格 / 樹內容重顯>進行重顯。`
	* 是
	* <font color=#a6a6a6>否</font>
