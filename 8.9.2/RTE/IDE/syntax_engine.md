# 語法
## 限制
* 於IDE上，使用`JavaScript`語法
* 於IDE上，只能使用引擎函數、方法 (不能使用`JavaScript`的函數、方法)
    > 註.因運算式可能同時在前端 / 後端執行

## 大小寫
* 不分大小寫
    > 噴碼會自動轉為區分大小寫

## 四則運算
* 常用項目
    | 符號 | 說明 | 
    | :--- | :--------- |
    | +	| 加 | 
    | -	| 減 | 
    | *	| 乘 | 
    | /	| 除 | 
    | %	| 取餘數 | 
    | +	| 字串相加 | 

## 比較運算子
* 常用項目
    | 符號 | 說明 | 
    | :--- | :--------- |
    | == | 等於 |
    | != | 	不等於 |
    | >	| 大於 |
    | <	| 小於 |
    | >= | 	大於等於 |
    | <= | 	小於等於 |

## 邏輯運算子
* 常用項目
    <table>
        <thead>
        <tr>
        <th align="left">符號</th>
        <th align="left">說明</th>
        </tr>
        </thead>
        <tbody><tr>
        <td align="left">!</td>
        <td align="left">斥</td>
        </tr>
        <tr>
        <td align="left">&amp;&amp;</td>
        <td align="left">且</td>
        </tr>
        <tr>
        <td align="left">||</td>
        <td align="left">或</td>
        </tr>
        </tbody>
    </table>    

## 字串表示: "abc"   
* 字串裡使用特殊字元
    | 符號 | 寫法 |
    | :--- | :--- |
    | 雙引號(") | \\" |
    | 斜線(\\) | \\\\ |

## 保留字
* true
* false
* empty
* null
