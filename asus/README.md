# 版本說明

此版本為與華碩雲端整合之客製版本，目前以雲端寶盒專案當作唯一專案。
企業/帳號/組織等維護以API方式提供給華碩呼叫，而引擎功能需和最新版引擎功能同步。
MAE除了登入登出改以華碩的流程，其他功能則需同步
